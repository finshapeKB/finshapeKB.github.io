  3. Digital banking building blocks

    Finshape’s main product is a custom Digital Banking solution. “Custom” in this context refers to a specific configuration of the provided building blocks (functional capabilities) combined with developing front-end applications and selected add-ons from scratch. These building blocks are not standalone products but rather DBOS infrastructure elements, DBOS components, connectors (APIs), tools, and applications from which the final product can be assembled.

  4. Applications

    An application is a software package that performs a specific function directly for an end user. An application can be self-contained or a group of programs. The program is a set of back-end operations that runs the application for the user. Applications use the computer's operating system (OS) and other supporting programs, typically system software, to function. An application requests services from and communicates with other technologies via an application programming interface (API).
