Task Steps

Task step is one simple action that in combination with other actions leads to the task completion.

Step is more appropriate naming as far as it's neutral and understandable for both common business and technical personas.

Steps (functions) may be undertaken in any types of tasks. We divide all possible steps to following types:

  • Admin Operation Functions

  • Security Operation Functions

  • Business Operation Functions

  • Communication Operation Functions

  • Report and Analysis Operation Functions

  • Management Operation Functions

  • DBOS Maintenance Operation Functions
