Components selection: PT04.02.01

Components selection for specific use-cases implementation is a part of Solution blueprint creation process. We need this information to be mentioned within the chapter DBOS Platform Components.

Step by step guidance on how to select DBOS components, that participate in a specific use-case .

  1. Find out with client what Role handles the task and what Category the task is.

  2. Go to Knowledge Base section Roles to review tasks that are assigned to this Role
    Note:
    We recommend you to open the Role's task list in a new tab to comfortably switch between this guide and needed sections

  3. Find within the task list the task title client is talking about

  4. Select task title within the list and main Task page will be opened

  5. Select Consists of steps to open the table, where you'll se what DBOS components correspond to task's steps

  6. Use Components' PID to include them to the Solution Blueprint, chapter DBOS Platform Components
    Note:
    If we have such tool (interoperability), the document line will be generated automatically.

  7. Go to the next Task.
