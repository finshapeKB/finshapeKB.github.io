  • Workspaces

    Workspace is a desktop or mobile page that is opened as bank's employees or clients log in to their banking application. Workspace may be designed for all Roles at the same time, or for a specific Role to provide handling only Role specific Tasks. Workspace usually comprise Introduction or Overview GUI page.

    • Workspaces for Bank's Employee (WAC)

      Bank's Employee workspace is a desktop page that is opened as bank's employees log in to their banking application. Bank's Employee manage their banking operations using Roles, so each role may have it's own workspace.

    • Workspaces for Corporate Client (Business)

      Bank's Corporate Client workspace is a desktop or a mobile page that is opened as bank's corporate clients log in to their internet banking application. Corporate clients manage their banking operations using Roles, such as bookkeeper, CEO, CFO etc., so each role may have it's own workspace.

    • Workspaces for Private Client (Retail)

      Workspace for Private Client is a desktop or a mobile UI starting page that is designed for Private Client role. The workspace is opened as Private Clients login to the internet banking application. Private Clients manage their banking operations within workspace.

        • Desktop workspace UI page for Private Client

          Desktop workspace for Private Client is a user interface starting page that is designed for Private Client role and runs on all desktop platforms. The workspace is opened as Private Clients login to the internet banking application. Private Clients manage their banking operations within workspace.

        • Desktop tasks UI pages for Private Clients

          Clients handle internet banking operations within specially designed UI pages. Those banking operations we considered for Tasks. Changing card pin code, making new payment, setting payment orders etc. are banking Tasks. Usually to gain a result you need to pass some steps. To ease handling all task steps, special UI page is created.
